Ryzen C6 Fix
​[English]
This repository contains a script to fix the freezing issue on AMD Ryzen 1000 series processors on Linux systems.
​The fix-ryzen.sh script is sufficient to apply the fix.
​The original ZenStates-Linux files are also included for those who prefer to use them directly.
​ 



\

​[العربية]
هذا المستودع يحتوي على سكريبت مخصص لحل مشكلة "تجمد النظام" (Freezing) في معالجات AMD Ryzen من الجيل الأول (Ryzen 1000 series) على أنظمة لينكس.
​ملف السكريبت fix-ryzen.sh يكفي تماماً لتطبيق الحل.
​تم تضمين الملفات الأصلية لأداة ZenStates-Linux أيضاً لمن يرغب في استخدامها بشكل مباشر.
